# SleepRadar App

Install SleepRadar from Home Assistant's Install app screen (Settings > Apps >
Install app) after adding the repository URL.

Set your Aqara Home app account, region, and FP2 `subject_id`, then start the
app. It publishes five MQTT discovery sensors under the configured
node id, which defaults to `aqara_fp2_sleep`.

Read the repository `README.md` before opening an issue. It explains how to
redact account details and device IDs.
